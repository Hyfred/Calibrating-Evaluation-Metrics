Calibrating Model-Based Evaluation Metrics for Summarization

---
Code for the paper: Calibrating Model-Based Evaluation Metrics for Summarization.

This repo builds on the ICLR 2025 paper "QA-CALIBRATION OF LANGUAGE MODEL CONFIDENCE SCORES":
- Paper: https://arxiv.org/pdf/2410.06615v2
- Supplementary repo: https://openreview.net/attachment?id=D2hhkU5O48&name=supplementary_material

Tasks and entry points

QA
- Data acquisition scripts live in src/data_acquisition/.
- Run experiments from experiments.py.

Summarization
- Run experiments from the Summarization/ directory (see Summarization/experiments.py).
